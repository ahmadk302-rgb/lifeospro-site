# Module 3: Project Command Center

## Purpose
Bridges the gap between the Architect's high-level quarterly milestones and concrete, executable projects with timelines, dependencies, and cost plans.

## Project Structure
Each project includes: name, linked goal, why it matters, outcome definition, start and target dates, status (Idea, Planned, Active, Waiting, Completed, Archived), priority, milestones, cost plan, next moves, dependencies, and blocks.

## Integration with Other Modules
Projects link to Architect goals via linked_goal_id. Cost plans feed into Tempo's budget targets. Active project milestones surface in the Commander's weekly review as candidates for the Big 3 tasks.
